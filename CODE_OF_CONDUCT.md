# Code of Conduct

## Our Commitment

This project aims to provide a respectful, harassment-free, and collaborative environment for all participants.

We welcome contributors of all backgrounds and expect everyone involved in the repository, reviews, and mentorship discussions to act professionally and constructively.

## Expected Behavior

- Be respectful in communication and review feedback.
- Focus on technical issues, not personal attacks.
- Accept feedback in good faith.
- Help create an inclusive and collaborative learning environment.

## Unacceptable Behavior

- Harassment, discrimination, or abusive language
- Personal attacks or intimidation
- Publishing private information without permission
- Disruptive or hostile conduct during collaboration

## Enforcement

Project maintainers may remove, edit, or reject contributions and participation that violate this code of conduct.

## Reporting

Report concerns to the repository maintainers through a private channel when available.

This document is adapted from the Contributor Covenant spirit, simplified for this repository.
